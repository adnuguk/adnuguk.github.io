#ADNUGUK – September 2011 Meeting – Completed

Oct 1st, 2011 2:43 pm | Comments

On Thursday 29th September 2011, the Aberdeen Developers .Net User Group was fortunate enough to welcome Phil Leggetter (@leggetter) of Pusher to do a talk entitled “Why the realtime web isn’t just another buzz phrase and why we should give a tweet”

If you were not able to attend the session you can find the slide deck that was used during the presentation here.

In addition, if you have any comments that you would like to make about the presentation, Phil has set up a feedback section on SpeakerRate, which you can find here. As always, feedback for these events is always welcome, and I would strongly encourage everyone to provide feedback.

If you have any other comments/suggestions about the User Group, then feel free to get in touch with me via my blog, or using the contact page.
